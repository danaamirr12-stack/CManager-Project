Det här är mitt kundhanteringssystem som jag gjort i C#. Det är en konsolapplikation där man kan skapa, visa, uppdatera och ta bort kunder via en meny. Projektet är uppbyggt med trelagers-struktur så att allt är uppdelat i olika delar - data, logik och presentation.
Programmet har ganska basic funktioner men täcker det som behövs. Man kan lägga till nya kunder, visa alla som finns sparade, uppdatera kundernas info och ta bort de som man inte behöver längre. All data sparas i JSON-filer så att allt ligger kvar mellan gångerna man kör programmet. Jag har även gjort unit tests för att se till att koden funkar som den ska.
I projektet har jag delat upp allt i olika mappar - Models för själva kundklassen, Repositories som sköter datahanteringen, Services för logiken, Presentation för konsol-menyn och Tests för alla tester. Data ligger i JSON-format och det finns unit tests med.
Denna uppgift var betydligt tuffare än föregående projekt. Den stora skillnaden var att man inte kunde se resultatet av sina ändringar direkt när man jobbade, till skillnad från webbutveckling där man ser förändringar i browsern direkt. Att arbeta med en konsolapplikation krävde mer debugging och testning för att säkerställa att allt funkade som det skulle.
